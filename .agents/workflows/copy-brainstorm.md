# /copy-brainstorm — ClearFlow Copy Brainstorm Workflow

> **Pre-requisite**: Read `.agent/copy-agent.md`, `.agent/brand/voice-and-tone.md`, and `.agent/brand/context.md` before starting.

---

## Step 1: Context Setting

Ask the user:

```
🧠 בואו נחשוב ביחד. ספרו לי:

1. מה המטרה? (למשל: כותרת חדשה ללנדינג, רעיונות לקמפיין, זווית למאמר)
2. יש משהו שכבר ניסיתם ולא עבד?
3. יש מתחרה שאהבתם את הקופי שלו? (שתפו דוגמה)
4. איזה רגש רוצים לעורר? (סקרנות / דחיפות / הזדהות / ביטחון)
```

> **Note**: This is a free-form conversation. Not all questions need answers. Accept whatever the user provides.

---

## Step 2: Concept Generation (🎨 יצירת קונספטים)

Generate **5-8 copy concepts**. Each concept is a mini-pitch, not a full copy.

### Output format:

```markdown
## 🎨 קונספטים — [נושא/מטרה]

### קונספט 1: "[שם הקונספט]"
**זווית:** [מה הגישה?]
**דוגמת Hook:** "[משפט אחד שמייצג את הקונספט]"
**למי מתאים:** [ערוץ / קהל / מצב]

| ייחודיות | ישירות | פוטנציאל המרה |
|:---------:|:-------:|:-------------:|
| ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ |

---

### קונספט 2: "[שם הקונספט]"
...
```

### Concept diversity rules:
- At least 1 concept that's **provocative/bold**
- At least 1 concept that's **empathetic/soft**
- At least 1 concept that's **data-driven/logical**
- No two concepts should use the same emotional lever

---

## Step 3: Selection + Direction

```
🎯 אילו קונספטים דיברו אליכם?

בחרו מספרים (אפשר יותר מאחד), ואגיד:
- איך לפתח אותם לקופי מלא
- לאיזה ערוץ הכי מתאים
- מה הצעד הבא

💡 אפשר גם לערבב — לקחת את ה-Hook מקונספט 2 עם הזווית של קונספט 5.
```

---

## Step 4: Bridge to /copy

Once the user picks winners:

```markdown
## 🔗 הצעד הבא

הקונספט שנבחר: **[שם]**
הזווית: [תיאור קצר]

כדי לפתח את זה לקופי מלא, הפעילו את `/copy` ואני אשתמש בקונספט הזה כבסיס.

או — רוצים שאפתח עוד וריאנטים של הקונספט הזה לפני שמתקדמים?
```
